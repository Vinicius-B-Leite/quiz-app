

## Quiz App
Quiz desenvolvido em react native com o objetivo de obter nota bimestral na matéria programação mobile da Etec. Futuramente será adicionado a função de criar categorias de quiz.

 - react-native 0.69.5
 - styled-components ^5.3.5
